Exucution Instructions
------------------------------------------------------------------------------------------

To execute program in interactive mode, use the following command:

python3 start.py interactive input2.txt computer-next/human-next depth

To execute program in one move mode, use the following command:

python3 start.py one-move input2.txt output2.txt depth

File Structure
------------------------------------------------------------------------------------------

The main function is in start.py file
The minmaxtree.py file contains the Minimax class which implements the minimax algorithm using three functions as given in the slides.
The gameboard.py file is exactly as the MaxConnect4Game.py from the sample code.

Output files
------------------------------------------------------------------------------------------
file1.txt stores the last state of the game for player1
file2.txt stores the last state of the game for player2
